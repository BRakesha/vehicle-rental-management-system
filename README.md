# Vehicle Rental Management System

## Project Overview

This project is a Django-based vehicle rental management system developed to manage vehicle booking and availability operations. The application allows users to search and book vehicles while enabling administrators to manage vehicle details through CRUD operations.

---

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

---

## Features

* Vehicle Add, Edit, Delete Operations
* Vehicle Availability Management
* Vehicle Booking System
* Search Functionality
* Vehicle Filtering Based on Name and Seating Capacity
* Automatic Vehicle Status Update After Booking
* Admin Vehicle Management

---

## Modules

### User Side

* View Vehicles
* Search Vehicles
* Book Vehicles

### Admin Side

* Add Vehicles
* Edit Vehicle Details
* Delete Vehicles
* Maintain Vehicle Availability
